
# Tree Finder

It's my first attempt at creating a small Sequential CNN model from scratch using Keras, as of now, the model can identify whether a tree is neem or banyan.

I could have added more variety of trees but I couldn't find a proper dataset and scraping the web for 500 images each for 2 of the many varieties of trees already took long enough. But I may expand on its capabilities in the future if time permits.

## Modules Used

- Keras (for model training and prediction)
- numpy (for making image matrices and number arrays)
- pillow (for image resizing and conversion)
- os (accessing training data and file manipulation)
